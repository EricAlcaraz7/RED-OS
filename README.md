# RED Operating System (REDos)

Este es **REDos**, nuestro sistema operativo creado con C# y Cosmos. Lo hemos hecho para aprender cómo funciona un ordenador desde dentro. 

## ¿Qué es REDos?
Es un sistema operativo sencillo que funciona con comandos de texto. Con él puedes limpiar la pantalla, ver nuestra versión personalizada del logo, usar una calculadora completa.

---

## ¿Qué comandos puedes usar?
Cuando entres en **REDos**, escribe estos comandos para probarlo:

* `help`: Te enseña la lista de cosas que puedes hacer.
* `calc`: Abre nuestra calculadora (¡suma, resta, multiplica, divide, saca restos y raíces cuadradas!).
* `logo`: Vuelve a mostrar nuestro logo rojo en arte ASCII.
* `clear`: Limpia toda la pantalla.
* `reboot`: Reinicia la máquina.
* `shutdown`: Apaga el sistema.

## Herramientas que hemos usado
Para hacer este proyecto hemos necesitado:
* **Lenguaje:** C# (el idioma en el que escribimos el código).
* **Cosmos SDK:** El "kit" que nos permite crear el sistema.
* **Visual Studio:** El programa donde escribimos todo.
* **VMware:** El simulador donde vemos cómo arranca nuestro SO.

---

## Cómo probarlo en tu PC
1.  Asegúrate de tener **Visual Studio** y el **Cosmos SDK** instalados.
2.  Descarga nuestra carpeta de proyecto.
3.  Abre el archivo del proyecto y dale al botón de **"Start"** (la flecha verde).
4.  ¡Espera unos segundos y verás nuestro logo rojo!

---

## ¿Cómo está organizado el código?
* **Kernel.cs**: Es el archivo principal. Aquí es donde sucede toda la magia (comandos, lógica y calculadora).
